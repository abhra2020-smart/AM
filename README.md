# FlowLang
The FlowLang programming language.                                                                                                                                               

This language now supports basic math.                                                                                                                                           

Supported operators:                                                                                                                                                             
`+`                                                                                                                                                                             
`-`                                                                                                                                                                             
`*`                                                                                                                                                                             
`/`                                                                                                                                                                             
`|` (integer division)                                                                                                                                                           
`%`                                                                                                                                                                             
`^` (power)                                                                                                                                                                     

Brackets are also supported.
