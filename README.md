# FlowLang
The FlowLang programming language.                                                                                                                                               

This language now supports basic math and booleans.                                                                                                                             

Supported operators:                                                                                                                                                             
`+`                                                                                                                                                                             
`-`                                                                                                                                                                             
`*`                                                                                                                                                                             
`/`                                                                                                                                                                             
`|` (integer division)                                                                                                                                                           
`%`                                                                                                                                                                             
`^` (power)                                                                                                                                                                     
`e` (equals)                                                                                                                                                                     
`l` or `L` (left shift)                                                                                                                                                         
`r` or `R` (right shift)                                                                                                                                                         
`>`                                                                                                                                                                             
`<`                                                                                                                                                                             
Brackets are also supported.
